IF-THENルール設計テンプレート

書き込むだけで「考えなくても動ける仕組み」が完成します。
トリガー（IF）と行動（THEN）を決めるだけでOKです。

💡 入力すると下の「マイIF-THENルール一覧」にリアルタイムで反映されます。まず1つだけ作ってみてください。

IF-THENルールとは？

IF（トリガー）
〇〇をしたら
→
THEN（行動）
△△をする

「やるかどうか」を毎回考えるのをやめる仕組みです。毎日すでにやっている習慣（歯磨き・食事・起床など）にセットするだけで、考えなくても動ける状態が作れます。まず1つだけ作ることから始めてください。

📝 参考例（タップで入力欄にコピー）

気に入ったものをタップすると、ルール①に自動入力されます
歯磨きをしたら　→　1行だけ書く（タイトルだけでOK）
就寝前の習慣に紐づける定番パターン
コーヒーを淹れたら　→　noteのエディタを開く（書かなくてもOK）
朝の習慣に紐づける。開くだけでOKにするのがポイント
仕事が終わったら　→　タイトルと見出しだけ作る（5分以内）
退勤という明確なトリガーを使う
電車に乗ったら　→　メモアプリを開いてアイデアを1つだけ書く
通勤時間をインプットではなくアウトプットに変える
シャワーを浴びる前に　→　今日やったことを1行だけ記録する
振り返り習慣を無理なく続けるパターン
1
まず最初の1つを作る（最重要）
未入力
IF　トリガー 毎日確実にやっている習慣を選ぶ
THEN　行動 5分以内・絶対できるレベルに設定する
メモ 終了ラインや注意点など（任意）
2
2つ目のルール（1つ目が定着してから）
未入力
IF　トリガー 1つ目と別の習慣を選ぶと効果的
THEN　行動 1つ目と別の行動を設定する
メモ 終了ラインや注意点など（任意）
3
3つ目のルール（余裕が出てきたら）
未入力
IF　トリガー 週1回など頻度を下げてもOK
THEN　行動 まとめ作業や振り返りに使うのもおすすめ
メモ 終了ラインや注意点など（任意）
